Trouve ton campus

Accueil

Cette application a été créée par l'association StrasWeb afin de donner un exemple d'utilisation des différentes données open data de la Communauté urbaine de Strasbourg.

  • Contributeurs :
  • Pierre Rudloff
  • Anne-Catherine Distelzwey
  • Bibliothèque utilisées :
  • jQuery
  • jQuery Mobile
  • PhoneGap
  • Leaflet
  • Données utilisées :
  • Emplacement et disponibilité des stations Vélhop (CTS)
  • Emplacement des stations Auto'trement (CUS)
  • Emplacement des parkings publics (CUS)
  • Emplacement des campus de l'Université de Strasbourg (Wikipédia)
  • Emplacement des stations de Tramway (Wikipédia)
  • OpenStreetMap

Certaines de ces données utilisent le système de projection Lambert 1 Nord (EPSG:27561). Pour les besoins de cette application, elles ont été converties au système de projection EPSG:4326.

Vous pouvez télécharger les données converties ainsi que le code source de l'application ici.
